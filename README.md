# Portfolio (Nuxt 3)

Site pessoal com projetos e estudos de caso, usando Nuxt 3 + i18n.

## Setup

```bash
npm install
```

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
npm run format
npm run i18n:check
```

## Conteúdo e i18n

- Traduções em `locales/pt.json`, `locales/en.json`, `locales/es.json`
- Links e dados globais em `data/site.ts`

## Deploy

O deploy é automático no Vercel quando há `git push` para o branch monitorado.
